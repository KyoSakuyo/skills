## 输入/输出

- 输入：`ocr-result/` 文件夹内的 `序号.md` 文件（每个序号对应书的一页）。
- 输出：合并为单一文件 `0.rough.md`，写入 `merge-result/`（不存在就创建）。

## 脚本参考

- 使用 `scripts/merge_rough.py` 完成合并。
- 默认按当前工作目录推导路径：
  - 输入：`<当前目录>/ocr-result`
  - 输出：`<当前目录>/merge-result/0.rough.md`
- 可选参数：
  - `--base-dir` 指定书籍目录
  - `--input-dir`/`--output-dir` 指定完整输入/输出目录
  - `--output-file` 指定输出文件名（默认 `0.rough.md`）
  - `--base-dir-from` 使用 UTF-8 文本文件提供书籍目录（首个非空行）
  - `--input-dir-from`/`--output-dir-from` 使用 UTF-8 文本文件提供输入/输出目录（首个非空行）
  - `--output-file-from` 使用 UTF-8 文本文件提供输出文件名或路径（首个非空行）
  
示例：

```bash
python .agent/skills/pdf-set/scripts/merge_rough.py --base-dir "C:\path\to\book"
```
