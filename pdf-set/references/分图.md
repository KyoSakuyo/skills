# 分图

## 目标
对“书籍名”执行分图任务时，自动定位书籍目录下的“图片”子目录（若不存在则创建），从 0 开始按 144dpi 输出图片。

## 操作步骤
1. 确认书籍目录存在 PDF 文件，默认路径为 `书籍名/书籍名.pdf`。
2. 确保输出目录为 `书籍名/图片`（不存在则创建）。
3. 适当修改下面的脚本后，使用脚本执行分图：

```bash
python3 .agent/skills/pdf-set/scripts/convert_pdf_to_images.py "书籍名/书籍名.pdf" "书籍名/图片" --dpi 144 --start 0 --format jpg
```

## 说明
- 必须从 0.jpg 开始命名。
- 输出分辨率固定 144dpi。
- 输出格式为 jpg。
