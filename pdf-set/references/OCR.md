CRITICAL: 按顺序执行以下阶段，勿跳步。

## 输入/输出
- 输入：`图片/` 文件夹内的图片，一次最多处理 5 张，不要跳页。
- 输出：创建 `OCR结果/` 文件夹；每张图生成一个 `原图序号.md`（即原图片的序号或文件名序号），即使无文字也要生成空文档，保持一一对应。
- 严禁使用浏览器；或者tesseract进行批量 OCR 生成；直接用模型视觉能力识别图片文字。

## 阶段 1：OCR 识别规则
1. **字符忠实**：简体保持简体，繁体保持繁体；不要做简繁转换。
2. **不补写**：断句没断完时不要补充或改写原文。
3. **版式剔除**：删除页眉/页脚页码、边码/边栏数字、每页重复的章节标题。
4. **复杂图形处理**：
   - 简单数学/希腊字母可用 LaTeX，例如：`($S(\\cancel{A})$)`, `\\sigma`, `$p^o$`。
   - 复杂公式/图示/配图无法文本化时，**跳过内容**，但在对应位置插入 `🀄，页码` 标记。（要注意，页码应该是在图片中出现的页眉或者页尾的页码。）
   - **禁止**使用 Markdown 图片语法；只用 `🀄，页码`。

## 阶段 2：Markdown 排版规则
1. **分段**：段落间用空行；同段内不要空行。
2. **标题**：章节标题或小标题一律用二级标题 `##`；禁止使用一级或三级标题。
3. **文本样式**：所有加粗/斜体/特殊样式一律还原为纯文本。
4. **首行缩进**：识别到另起的新段落时，新段落段首加两个空格字符的首行缩进；所有新段落都需要。若判断该段为上页结束处断开的段落之续接部分，则不要加首行缩进。

## 阶段 3：脚注转移（仅页尾注）
1. 若页尾有脚注，且正文中有对应文内注号：
   - 将页尾注释插入对应文内注号位置；
   - 用 `<sup>***【脚注全文】***</sup>` 包裹；
   - 删除正文中的文内注号；
   - 删除页尾脚注区。
2. 如果遇到注释中一条注释的末尾是「接下页」或者是一个「-」符号，而且你判断话没说完的情况的情况，通常情况下这是一个没有写完的注释，你需要在注释的最末尾加一个🀄的符号；它的下一页会有一条没有出现标号的另一个看起来话没说完的注释（可能会有「续上页」的文字提示），你需要把它放在该页的最前端，同样也需要加一个🀄的符号。
3. 若正文中的注号对应**章节末/书末注释**：
   - 不删除文内注号；
   - 将该注号改为 `<sup>***【】***</sup>` 形式；
   - 章节末/书末注释列表保持原样，不做内嵌或清理。
4. 不同注释之间保留空行（用于章节末/书末注释列表）。

## 阶段 4：清理项（必须删除）
- 正文页码（页眉/页脚）。
- 边码 / 边栏编号。
- 每页重复出现的章节标题名。
- 正文内**对应页尾注释**的文内注号。
- 已完成内嵌后的页尾注释块。

## 阶段 5：输出检查
- 每张图都有对应 `原图序号.md`；
- `🀄，页码` 仅用于无法文本化的位置；
- 段落与标题格式符合上述规则；
- 无 Markdown 图片语法。
