## 阶段 2：OCR 识别规则
1. **字符忠实**：简体保持简体，繁体保持繁体；不要做简繁转换。更不要给原文换语言，不要把原书中的「他」换成「he」，把「和」换成「and」，把「的」换成「of」或者「의 」，不要把「了」换成「ました」…… 与此同时，保持原文的完整性，不可以增减掉和改写任何该页正文的内容。
2. **不补写**：如果该页结尾断句没断完，不要**补充**或改写原文。**不允许从下一页调取内容来补写上一页**。这很重要！！识别到什么就输出什么。不要看到页尾结束在「小白」就加一个「菜」输出「小白菜」；也不要看到页首开始在「白菜」就加一个「小」输出「小白菜」。
3. **版式剔除**：删除页眉/页脚页码、边码/边栏数字、每页重复的章节标题。
4. **复杂图形处理**：
   - 简单数学/希腊字母可用 LaTeX，例如：`($S(\\cancel{A})$)`, `\\sigma`, `$p^o$`。
   - 当复杂公式/图示/配图无法文本化时，**跳过内容**，但在对应位置插入 `🀄️，该页页码` 标记。（要注意，页码应该是在图片中出现的页眉或者页尾的页码。）
   - **禁止**使用 Markdown 图片语法；只用 `🀄️，该页页码`表示图片。
   - 注意，`🀄️，该页页码`只是提示符，提示该位置有图片，以便我在正文中手动插入图片。

## 阶段 3：排版规则
1. **分段**：段落间用**两个换行符制作空行**；而同一个段落内不要空行──也不许用一个换行符换行。
2. **标题**：章节标题或小标题一律用两个井号的二级标题 `##`；禁止使用一级或三级标题。
3. **文本样式**：所有加粗/斜体/特殊样式一律还原为纯文本。
4. **首行缩进**：首先判断是不是正文部分；如果是：你需要识别这是不是一个段落的段首（段首通常会在图片中有首行缩进的效果），如果是：新段落**段首**加 两个空格字符的首行缩进；注意，只有段首需要，而不是给每一行都加首行缩进。若判断该段为上一页结束处断开的段落之续接部分（续接部分通常会在图片中**没有**首行缩进的效果，而且句意通常都没有说完，不完整，你尤其需要在一张图片的开头部分特别注意是不是续接部分），则不要加首行缩进。

## 阶段 4：脚注转移（仅页尾注）
1. 搜寻文内的文内注号和页尾的脚注的注号──若页尾有脚注，且正文中有与其注号对应的文内注号：
   - 将页尾注释插入对应文内注号位置；
   - 用 `<sup>***【脚注全文】***</sup>` 包裹；
   - 删除正文中的文内注号；
   - 删除已完成内嵌后的页尾注释块（包括页尾注号）。
   注意：注释中不可以分段；即使原图中的注释是分段的，你就把它们合并为一段。
2. 如果遇到注释中一条注释的末尾是「接下页」或者是一个「-」符号，而且你判断话没说完的情况的情况，通常情况下这是一个没有写完的注释，你需要在注释的最末尾加一个🀄️的符号；它的下一页会有一条没有出现标号的另一个看起来话没说完的注释（可能会有「续上页」的文字提示；或者没有任何的注号），你需要把它放在该页的最前端，同样也需要加一个🀄️的符号。
3. 若正文中的注号对应**章节末/书末注释**：
   - 不删除文内注号；
   - 将该注号改为 `<sup>***【】***</sup>` 形式；
   - 章节末/书末注释列表保持原样，不做内嵌或清理。
4. 不同注释之间保留空行（用于章节末/书末注释列表）。
