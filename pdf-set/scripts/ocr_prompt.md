如果是无任何文字内容的空页，你就留下一个使用`\n`表示的空行就可以了。不用输出其他内容。
## 正文字符边界（核心）
- 每张图片都要明确识别正文的第一个字符与最后一个字符（包含断句残句）。
- .md 必须从起始字符写到结束字符，不补不删。
- 严禁：猜测、推断、补全句意。
- 例：若 图片内容的 的正文结束字符是「…身体的延」；
  则 输出的文本 必须以「身体的延」结尾；不得以猜想它后面可能是「续」或「伸」等字而补全。
- 例：若图片内容的最后是「。无」，则 输出的文本 必须以「。无」结尾；不得以把单独的「无」删掉。
  若图片的开始是以「中一群人……」你不得认为「中」是一个多余字符而删掉它。

### 断句处理
- 结尾为「，」「、」「：」等明显未完结符号时：保持原样输出，不得续写。
- 识别到什么就输出什么；完全尊重图片原始文字边界。
- 不改写、不补全、不润色、不推断。
- 如果该页的结尾只有半句，也要输出，不要漏掉！！

## 清理（必须删除）
- 页眉页脚（包括小标题，书目，页码）
- 边码/边栏编号
- 不在正文中（和正文有距离），而是在每页页眉中的章节标题名

## OCR 识别规则
- 字符忠实：简体保持简体，繁体保持繁体；不做简繁转换；不换语言（不要把「他/和/的/了」换成外语词）。
- 版式剔除：删除页眉/页脚页码、边码/边栏数字、每页重复的章节标题。
- 复杂图形处理：
  - 简单数学/希腊字母可用 LaTeX，例如：($S(\\cancel{A})$), \\sigma, $p^o$。
  - 复杂公式/图示/配图无法文本化时，跳过内容，但在对应位置插入 🀄️，该页页码 标记。（页码以图片页眉/页尾为准）
  - 禁止使用 Markdown 图片语法；只用 🀄️，该页页码。
  - 🀄️，该页页码 只是提示符，便于后续手动插图。

## 排版规则
### 分段规则
- 分段：段落间用两个换行符`\n\n`换行；
- 段内不用`\n`空行，连续输出所有内容。
- 每页输出结尾不留空行。

  错误示例1：

   ```
     恶与罪在暴民心理中的吸引力早已有之。暴民一向会欢迎“暴
       
   ```
  错误示例2：

   ```
     恶与罪在暴民心理中的吸
     引力早已有之。暴民一向会欢迎“暴
       
   ```
  需要以：

   ```
     恶与罪在暴民心理中的吸引力早已有之。暴民一向会欢迎“暴
   ```

   的形式直接结束该页。

- 标题：章节标题或小标题用二级标题 ##；禁止一级/三级标题(目录中的标题不用加##)
- 文本样式：加粗/斜体/特殊样式一律还原为纯文本。


### 首行缩进：
  - 以内容的连贯性和排版的空行位置判断该段落是否是新段落，新段段首加两个空格首行缩进。
  - 如果是一连串注释或者正文中列举，或者类似于参考资料中的人名或参考书目列举，你统一给每个不同内容之间都加新行，并且做首行缩进。
  - 若为上一页断段续接（通常无首行缩进且句意未完），则不要缩进。
#### 图片开头段落处理（格外注意）
  - 检查该页图片的开头是否有首行缩减的两个空格，如果它有，则说明它是新段落的开始，必须加首行缩进。
  若没有，判断该段为上一页结束处断开的段落之续接部分──续接部分通常会在图片中**没有**首行缩进（空两个空格的效果），而且句意通常都没有说完，不完整，没有完整主谓宾──则不要加首行缩进──对于这一点你可以处理地激进些，如果你不确定是否为续接部分，你就当它是续接部分，不要加首行缩进。
  - 对话体/诗歌/列表/特殊格式……：每人对话间留空行，每新行都缩进。
  - 不在正文内，单独成段的`🀄️，该页页码`也需要首行缩进
  -  不在正文内，单独成段的<sup>标签也要首行缩进

示例：

```
……上一段落结尾
（空行）
  下一段落开头
```
### 注释转移

#### 处理边注 Marginal Notes：转移
- 若注释类型是marginal notes（边注）──即，正文内容是大号字体，注释内容用小字字体在正文内穿插，把注释内容用 <sup>【】</sup> 包裹即可。

#### 处理尾注 Endnote：不转移
若页脚无注释内容但是正文有标号──则这是尾注 Endnotes类型的书籍：
  - 不删除文内注号；
  - 文内注号改为 <sup>【原注号】</sup> 形式；不加入任何注释内容。

处理章节末尾注的标准：
  - 尾注内容保持原样，不做内嵌或清理。
  - 注释间保留空行。
  - 不需要sup标签

### 处理脚注 Footnote 和边注 marginal notes：脚注转移
- 若页脚有脚注：把脚注文本移到正文中对应注号后，用 <sup>【】</sup> 包裹，并删除原注号与页脚脚注。

- 注释不得分段；原图注释如分段，需合并为一段。
